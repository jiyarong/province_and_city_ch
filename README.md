## Installation
Add this line to your application's Gemfile:

```ruby
gem 'province_and_city_ch'
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install province_and_city_ch
$ rails g province_and_city_ch

```
